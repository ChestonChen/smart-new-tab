Theme

Stale-tab detection

Banner appears on the new tab page when tabs haven't been activated for the configured number of days. Pinned tabs are never flagged.

Impact stats

Smart New Tab tracks how many duplicates you've closed, tabs you've recovered, and bulk-close actions — purely on this device. Reset the counter any time.

Custom rules
